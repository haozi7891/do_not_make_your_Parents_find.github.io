## 🚀 快速开始

### 前提条件
- Linux系统（推荐Arch Linux/Ubuntu）
- GCC编译器
- ncurses库

### 安装依赖

**Arch Linux:**
sudo pacman -S gcc ncurses
**Debian:**
sudo apt-get install gcc libncurses5-dev
